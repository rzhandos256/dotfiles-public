dotfiles

Personal macOS development environment. We maintain dotfiles - a comprehensive configuration for modern development. It consists of: **Neovim** - text editor with language support **Alacritty** - terminal emulator **Tmux** - session management **Shell** - environment configuration **Karabiner** - keyboard customization **Homebrew** - package management ---

Getting started

git clone https://github.com/rzhandos256/dotfiles-public.git
Fork and adapt for your setup. ---

FAQ

**What are dotfiles?** Configuration files for development tools and applications. **Can I use these?** Yes, fork and customize. **What do I need?** macOS and basic terminal knowledge. **How do I customize?** Edit the config files for each tool. **Where can I contribute?** GitHub repository. ---
